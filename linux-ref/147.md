
# Linux mkbootdisk命令



Linux mkbootdisk命令用于建立目前系统的启动盘。

mkbootdisk可建立目前系统的启动盘。

### 语法

```
mkbootdisk [--noprompt][--verbose][--version][--device <设备>][--mkinitrdargs <参数>][kernel 版本]
```

**参数**：

*   --device&lt;设备&gt; 指定设备。
*   --mkinitrdargs&lt;参数&gt; 设置mkinitrd的参数。
*   --noprompt 不会提示用户插入磁盘。
*   --verbose 执行时显示详细的信息。
*   --version 显示版本信息。
